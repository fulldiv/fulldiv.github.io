FREGUENTLY ASKED QUESTIONS

— What is Full Stack Development?

A Full-stack Web Developer is a person that can work with both the front-end and back-end. Front-end refers to the part of an application that users can view and interact with, through the use of HTML, CSS, and JavaScript. The back-end portion of the application usually runs on a server and handles database interactions, user authentication, etc.

Full-stack developers can:

— Program a browser (with HTML, CSS, JavaScript, etc.)
— Program a server (with Node.js, or Python, etc.)
— Program a database (with MongoDB, or PostgreSQL, etc.)

Full-stack developers are proficient in every aspect of the web interworkings such as setting up and configuring servers and databases, writing server-side code and APIs, coding the graphical user interface and everything in between.
