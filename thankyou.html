We are going to cover full stack development, what it means and all the tools, languages, libraries and frameworks you need to learn to be able to develop websites, mobile and web applications, server side scripts and much more.



A full stack developer is a web developer who works with both the front and back ends of a website or application—meaning they can tackle projects that involve databases, building user-facing websites.

  • ― Front end developers build the visible parts of websites that users see and interact within their web browsers.

  • — The front end of a website (or web or mobile application) is the part a user sees and directly interacts with. The front end is built with languages like HTML, CSS and JavaScript

Back end developers build the “under the hood” parts of websites that users don’t interact with directly.

  • We will also use Javascript for back end development with Nodejs which is an environment to run javascript outside the browser box.

There are six main topics we will go through:

▶ HTML or Hypertext Markup Language, it is a type of markup language, It wraps, or “marks up” data within HTML tags, which define the data and describe its purpose on the webpage.


▶ CSS or Cascading Style Sheets is a style sheet language used for describing how a document written in a markup language such as HTML will look like, think of it as the makeup that makes HTML look beautiful.


▶ Javascript is the Programming Language for the Web although it can be used to develop mobile applications, server side code and much more. HTML and CSS are static meaning that You cannot change HTML and CSS after a website gets loaded, thats where Javascript comes to play, JavaScript gives you the ability to update and change the HTML and CSS on the page either automatically or through an interaction by the user such as a mouse click or hover. JavaScript can also calculate, manipulate and validate data.


▶ Libraries and frameworks add functionality to your code and make your life easier by simplifying complex problems so you can focus on finishing your project.


▶ Databases are used to organize and store data.


▶ Version control which is a system that records changes to a file or set of files over time so that you can recall specific versions later.


▶ API stands for Application Programming Interface, it is an interface or communication protocol that allows different parts of a computer program to talk to each other and exchange data.
