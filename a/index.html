What is  Full Stack? 

A full-stack developer is a web developer who works with both the front and back ends of a website or application—meaning they can tackle projects that involve databases, server code, and building user-facing websites.


- Front end developers build the visible parts of websites that users see and interact within their web browsers.

- The front end of a website is the part a user sees and directly interacts with. The front end is built with languages like HTML, CSS, and JavaScript

Back end developers build the “under the hood” parts of websites that users don’t interact with directly such as:

The code that runs on the server and APIs that allow servers to talk to each other under the hood.
